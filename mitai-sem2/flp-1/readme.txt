Ovladani projektu pomoci make
=============================
Preklad:
$ make

Spusteni projektu:
$ make run opt={-i|-1|-2} [file=filename]

Spusteni testu:
$ make test

Smazani generovanych souboru prekladem:
$ make clean

Zabaleni projektu k odevzdani:
$ make zip
